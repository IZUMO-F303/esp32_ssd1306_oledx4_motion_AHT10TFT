# ESP32 多機能環境時計 & 天気予報ステーション
## (4連OLED + ST7789 TFT + AHT10 + PIR + Touch)

このプロジェクトは、ESP32を核として、4枚のOLEDディスプレイと1枚の大型TFTディスプレイを組み合わせた、圧倒的な情報量を誇るデスクトップ情報端末です。
NTPによる正確な時刻、OpenWeatherMapによる詳細な天気予報、およびAHT10センサーによる室内環境（絶対湿度を含む）を統合して表示します。

## 🌟 主な機能

### 1. 5画面独立表示システム
- **TFT (中央 - SPI)**: 
  - 巨大なデジタル時計を表示。
  - 室内環境データ（温度・湿度・絶対湿度）を表示。
  - 絶対湿度は健康指標に基づき色分け（緑：快適、黄：注意、赤：警戒）。
- **OLED 1 (左上 - Bus 0, 0x3C)**: 
  - 現在の日付、曜日、および月齢（数値とグラフィック）。
- **OLED 2 (右上 - Bus 0, 0x3D)**: 
  - 「昨日の天気」の概況と、過去24時間の最高・最低気温。
- **OLED 3 (左下 - Bus 1, 0x3C)**: 
  - 「今日の天気」の概況と、予想最高・最低気温。
- **OLED 4 (右下 - Bus 1, 0x3D)**: 
  - 「明日の天気」の概況と、予想最高・最低気温。

### 2. ダイナミック・グラフ表示
- **詳細な気温推移**: 
  - タッチ操作により、OLED 2〜4が「気温グラフモード」に切り替わります。
  - **過去24時間（履歴）** ＋ **未来48時間（予報）** の気温変化を可視化。
  - 30分ごとに現在の気温を記録し、NVS（不揮発性メモリ）に保存するため、再起動後もグラフが維持されます。

### 3. スマート省電力 & スクリーンセーバー
- **1分間無操作**: 全画面が「3Dスターフィールド（流れる星空）」スクリーンセーバーへ移行。
- **2分間無操作**: 全ディスプレイが完全消灯（パネルスリープ ＋ TFTバックライトオフ）。
- **バックグラウンド更新**: 消灯中も30分ごとの天気取得と時刻同期、履歴保存を継続。

### 4. インテリジェント・ウェイクアップ（動作分離）
- **人感センサー (PIR) 反応**:
  - スリープから復帰し、**通常情報モード**（昨日・今日・明日の天気）を表示します。通りかかるだけで必要な情報がすぐに確認できます。
- **タッチセンサー 反応**:
  - スリープから復帰、または点灯中に反応し、**20秒間の気温グラフモード**を開始します。詳細なデータを確認したい時のみ、能動的な操作でグラフを呼び出せます。

## 🛠 ハードウェア構成

### マイコン
- **ESP32** (DevKit V1 推奨)

### ディスプレイ
- **SSD1306 0.96インチ OLED (I2C)** x 4枚
  - **I2C Bus 0 (SDA:21, SCL:22)**: アドレス 0x3C, 0x3D
  - **I2C Bus 1 (SDA:17, SCL:16)**: アドレス 0x3C, 0x3D
- **ST7789 240x240 TFT (SPI)** x 1枚
  - **SPI (MOSI:23, SCLK:18, DC:2, RST:4, BL:5)**

### センサー・入力
- **AHT10 温湿度センサー**: I2C Bus 1
- **PIR人感センサー**: GPIO 14
- **タッチセンサー**: GPIO 33

## 🚀 セットアップ

1. **secrets.h の作成**: `src/` ディレクトリに以下の内容で `secrets.h` を作成してください。

```cpp
#ifndef SECRETS_H
#define SECRETS_H

const char* SECRET_SSID = "あなたのWiFi_SSID";
const char* SECRET_PASS = "あなたのWiFi_パスワード";
const char* SECRET_API_KEY = "OpenWeatherMap_API_KEY";
const char* SECRET_CITY = "Tokyo,JP";

#endif
```

2. **ビルドと書き込み**: 
   PlatformIO を使用してビルドし、ESP32に書き込んでください。

## 📚 使用ライブラリ
- **LovyanGFX**: 高速かつ高度なグラフィック描画。
- **ArduinoJson**: 天気APIのパース。
- **NTPClient**: 時刻同期。
- **Adafruit AHTX0**: センサー制御。
- **Preferences**: 履歴データの永続化。
