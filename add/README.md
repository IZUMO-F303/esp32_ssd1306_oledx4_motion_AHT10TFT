# ESP32 環境時計 (AHT10 + ST7789 + NTP + PIR)

このプロジェクトは、ESP32 デバイスを使用して NTP サーバーから正確な時刻を取得し、同時に AHT10 センサーから温湿度を取得して表示するデジタル時計兼環境モニタです。人感センサー（PIR）を搭載し、動きがない場合にはスクリーンセーバーの起動や自動消灯を行う省電力・保護機能を備えています。

## 主な機能
- **高精度デジタル時計**: 
  - NTP サーバーから日本時間 (JST) を取得。
  - 画面上部に大型フォント (`Font7`) で「時:分」を表示。
  - リソース保護のため、表示更新は **1分に1回** (分が切り替わるタイミング) に最適化。
- **温湿度測定**: AHT10 センサーによる温度（摂氏）と相対湿度の測定。
- **絶対湿度計算**: テトンスの式を用いて、温度と相対湿度から絶対湿度（$g/m^3$）を算出。
- **人感センサーによる表示制御**:
  - **通常モード**: 動きを検知している間は、時計と環境データを表示。
  - **スクリーンセーバー**: 1分間動きがない場合、液晶保護のため「スターフィールド（星間飛行）」アニメーションを表示。
  - **自動消灯**: 2分間動きがない場合、バックライトを消灯して省電力化。
  - **自動復帰**: 動きを検知すると即座に通常モードへ復帰。
- **視認性の高い表示 (240x240 ST7789 TFT)**: 
  - 時刻（白）、温度（オレンジ）、相対湿度（青）、絶対湿度（緑）を色分けして表示。
- **WiFi接続**: 起動時に自動的に指定のアクセスポイントへ接続し、時刻同期を開始。

## ハードウェア構成

### 使用部品
- ESP32 Development Board
- AHT10 温湿度センサー
- ST7789 TFT ディスプレイ (240x240)
- PIR 人感センサー (例: HC-SR501, SR602 など)

### ピンアサイン

#### AHT10 センサー (I2C)
| ESP32 Pin | AHT10 Pin | 機能 |
| :--- | :--- | :--- |
| GPIO 17 | SDA | データ |
| GPIO 16 | SCL | クロック |

#### ST7789 TFT ディスプレイ (SPI)
| ESP32 Pin | Display Pin | 機能 |
| :--- | :--- | :--- |
| GPIO 18 | SCL (SCLK) | SPI クロック |
| GPIO 23 | SDA (MOSI) | SPI データ |
| GPIO 2 | DC (RS) | データ/コマンド選択 |
| GPIO 4 | RES (RST) | リセット |
| GPIO 5 | BLK (LED) | バックライト |

#### PIR 人感センサー
| ESP32 Pin | PIR Pin | 機能 |
| :--- | :--- | :--- |
| GPIO 14 | OUT | 信号出力 |

## ソフトウェア環境

### 依存ライブラリ
- **LovyanGFX**: ディスプレイ制御およびフォント表示。
- **Adafruit AHTX0**: AHT10 センサーのデータ取得。
- **WiFi / time**: WiFi 接続および NTP 時刻同期。

### 設定項目
`src/main.cpp` 内の以下の変数を環境に合わせて変更してください：
- `ssid`: Wi-Fi の SSID
- `password`: Wi-Fi のパスワード
- `STARFIELD_TIMEOUT`: スクリーンセーバー開始までの時間（デフォルト1分）
- `DISPLAY_OFF_TIMEOUT`: 自動消灯までの時間（デフォルト2分）

## 使用方法
1. PlatformIO でプロジェクトを開きます。
2. 回路を接続し、ビルド・書き込みを行います。
3. 起動時、画面に `WiFi connecting...` と表示され、接続完了後にデジタル時計と環境データが表示されます。
4. しばらく動きがないと星のアニメーションが始まり、さらに時間が経過すると画面が消えます。センサーの前で動くと元の表示に戻ります。
